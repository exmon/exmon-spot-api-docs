# Official Documentation for the EXMON APIs

* Official announcements regarding API changes, maintenance, or downtime will be published via official EXMON channels.
* All endpoints, parameters, and payloads described in this repository are considered **official** and **supported**.
* The use of undocumented endpoints or parameters is **not supported** and is done at your own risk.
